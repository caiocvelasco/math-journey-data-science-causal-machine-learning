Phase 1: Logic & Set Theory

Motivation

Mathematical reasoning begins with logic. Proofs in analysis, probability, and statistics rely on the ability to manipulate statements rigorously, understand how they combine, and know when two statements are logically equivalent.

In causal machine learning, assumptions are often stated in logical form. For example:

If ignorability holds, then a treatment effect is identifiable.

Thus, before diving into analysis and probability, we establish a foundation in logic and set theory.

1. Sentential Logic

1.1 Statements and Connectives

A statement (or proposition) is a declarative sentence that is either true or false.

  • Example: “3 is even” (false), “Barcelona is in Spain” (true).

Logical connectives:

  • Negation: \(\lnot p\)
  • Conjunction: \(p \land q\)
  • Disjunction: \(p \lor q\)
  • Conditional: \(p \to q\)
  • Biconditional: \(p \leftrightarrow q\)

1.2 Truth Tables

A truth table shows how the truth value of a compound statement depends on its parts.

Example: Prove that an implication is equivalent to its contrapositive.

We want to show:

\[ p \to q \;\equiv\; \lnot q \to \lnot p \]

\(p\) \(q\) \(p \to q\) \(\lnot q\) \(\lnot p\) \(\lnot q \to \lnot p\)
T T T F F T
T F F T F F
F T T F T T
F F T T T T

Since the last two columns match, the implication is equivalent to its contrapositive. ∎

Definitions:
- A tautology is a statement that is always true.
- A contradiction is a statement that is always false.
- Two statements are logically equivalent if they have the same truth table.

2. Sets and Truth Sets

A set is a collection of distinct objects.
Example: \(A = \{2,4,6,8\}\).

The truth set of a predicate \(P(x)\) over a universe \(U\) is

\[ \{x \in U \mid P(x) \text{ is true}\}. \]

Example: If \(U = \mathbb{Z}\) and \(P(x): x^2 = 4\), then the truth set is \(\{-2, 2\}\).

Basic operations on sets (parallel to logical connectives):

  • Union: \(A \cup B \quad\leftrightarrow\quad p \lor q\)
  • Intersection: \(A \cap B \quad\leftrightarrow\quad p \land q\)
  • Complement: \(A^c \quad\leftrightarrow\quad \lnot p\)
  • Difference: \(A \setminus B = \{x \in A \mid x \notin B\}\)

3. Exercises

  1. Build a truth table for \((p \lor q) \land \lnot p\).
  2. Show that \((p \to q) \equiv (\lnot q \to \lnot p)\).
  3. Find the truth set of \(P(x): x^2 = 4\) over \(U = \mathbb{Z}\).
  4. Let
    \[ A = \{x \in \mathbb{Z} \mid x \text{ even}\}, \quad B = \{x \in \mathbb{Z} \mid x \text{ multiple of 4}\}. \]
    Prove or disprove: \(B \subseteq A\).
  5. Show with a truth table that \((p \land q) \to p\) is a tautology.

4. Connection to Causal ML

  • Logical connectives allow us to formalize assumptions.
  • Truth sets connect directly to sample spaces in probability.
  • Set operations mirror SQL/database operations (our application project).
  • The contrapositive law underlies many proofs in statistics and causal inference:

If ignorability holds, then the treatment effect is identifiable.
Equivalently, if the treatment effect is not identifiable, ignorability does not hold.
